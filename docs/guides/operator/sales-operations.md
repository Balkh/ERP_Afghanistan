# Operator/Cashier Guide

## Overview
This guide covers daily sales and invoice operations.

## Daily Start
1. Login with your username and password
2. Check dashboard for important alerts
3. Verify register/cash amount

## Making a Sale

### Quick Sale Workflow
1. Go to Sales → New Invoice
2. Search product by:
   - Name
   - SKU
   - Barcode (use scanner or type)
3. Enter quantity
4. Add more products as needed
5. Select payment method:
   - Cash
   - Bank Transfer
   - Mobile Money
   - Hawala
6. Click "Save & Print" or "Save"

### Barcode Scanning
- Use barcode scanner or keyboard
- Product automatically added
- Enter quantity if more than 1

## Common Tasks

### Finding Products
- Use search bar at top
- Scan barcode
- Browse categories

### Invoice Correction
- Only draft invoices can be edited
- For posted invoices, create return or adjustment

### Returns
1. Go to Sales → Returns
2. Select original invoice
3. Enter return items and quantities
4. Process refund

## Payment Methods

### Cash
- Most common
- Immediate settlement

### Bank Transfer
- Record bank reference number
- Verify with bank statement

### Mobile Money
- Record transaction ID
- Common for mobile transfers

### Hawala
- Traditional money transfer
- Record hawala operator name
- Record reference number

## End of Day

### Daily Summary
1. Check total sales
2. Verify payments received
3. Print daily report
4. Ensure all invoices saved

### Cash Reconciliation
1. Count actual cash in drawer
2. Compare with system records
3. Report any discrepancies

## Keyboard Shortcuts
- `Ctrl+N` - New invoice
- `Ctrl+S` - Save
- `Ctrl+P` - Print
- `F2` - Search product
- `Esc` - Cancel/Close

## Warning Section
⚠️ **Important:**
- Verify product prices before saving
- Always give customer copy of invoice
- Keep cash secure

## Common Issues

### Product Not Found
- Check spelling
- Try barcode
- Ask supervisor to add product

### Price Wrong
- Cannot change in invoice
- Must be corrected in product first

### Payment Failed
- Check payment method settings
- Contact IT support
