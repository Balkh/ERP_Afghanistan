# Quick Start Guide - Pharmacy ERP

## Getting Started (5 Minutes)

### 1. Login
- Username: `admin`
- Password: `admin123`
- **Change password immediately!**

### 2. Main Screen
- Left sidebar: Navigation menu
- Top: Search bar
- Center: Dashboard with alerts
- Right: Quick actions

### 3. Basic Workflows

#### Make a Sale
1. Sales → New Invoice
2. Add products (search or scan)
3. Select payment method
4. Save

#### Receive Products
1. Purchases → Receive
2. Select PO
3. Verify quantities
4. Confirm

#### Check Stock
1. Inventory → Products
2. View quantities and batches

#### View Reports
1. Accounting → Reports
2. Select report type
3. Set date range
4. View/Export
