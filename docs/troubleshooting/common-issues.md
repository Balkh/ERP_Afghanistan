# Troubleshooting Guide

## Common Issues and Solutions

## Login Issues

### Can't Login
1. Verify username and password
2. Check Caps Lock is off
3. Try default credentials (admin/admin123)
4. Clear browser cache
5. Restart application

### Forgot Password
- Contact system administrator
- Database access required for reset

## Inventory Issues

### Product Not Found
- Product may not be created yet
- Check spelling carefully
- Try barcode instead of name
- Contact admin to add product

### Stock Mismatch
1. Check batch quantities in Inventory → Batches
2. Review stock movements in Inventory → Movements
3. Verify no missing transactions
4. Create adjustment if needed

### Expiry Alert
- Check Dashboard for expiry warnings
- Review Inventory → Batches → Expiring Soon
- Prioritize selling near-expiry items
- Consider returns to supplier

## Sales Issues

### Invoice Won't Save
1. Ensure at least one product added
2. Verify payment method selected
3. Check required fields filled
4. Save as draft if posting fails

### Price Wrong
- Cannot edit price in invoice
- Edit product price first, then create invoice

### Can't Process Return
- Original invoice must exist
- Return must be within allowed period
- Quantity cannot exceed original purchase

## Payment Issues

### Payment Method Not Working
1. Verify payment method is active
2. Check account configuration
3. Contact administrator

### Hawala Reference Missing
- Hawala requires operator name
- Record reference number
- Verify with customer

## Accounting Issues

### Trial Balance Not Balancing
1. Check for unposted journal entries
2. Verify all entries balanced
3. Review recent transactions

### Report Shows Zero
1. Check date range
2. Verify transactions are posted
3. Check account selection

### Can't Create Journal Entry
1. Must have at least 2 lines
2. Debits must equal credits
3. Select valid accounts

## Currency Issues

### Exchange Rate Wrong
- Go to Accounting → Currencies
- Update exchange rate
- Rate applies to new transactions

### Currency Conversion Error
- Verify both currencies exist
- Check exchange rate is set
- Try using explicit rate

## Backup/Restore Issues

### Backup Fails
1. Check disk space
2. Verify write permissions
3. Check database not corrupted

### Restore Won't Work
1. Verify backup file exists
2. Check backup is not corrupted
3. Ensure same version
4. Contact support if needed

## System Issues

### Application Slow
1. Check internet connection
2. Close unused windows
3. Clear browser cache
4. Restart application

### Data Not Loading
1. Check database connection
2. Refresh page
3. Restart application

### Error Messages
- Take screenshot of error
- Note exact time of error
- Contact administrator with details
